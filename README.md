cproj-create is an individual module that let users to create C/C++ 
Projects from the terminal Itself! Useful if you don't want to 
create each individual file painfully.

# Usage - 

Assuming that you have installed the executable (bundled executable + 
config file. BOTH ARE IMPORTANT!), in terminal you can just 

`cproj-create new [projName]`

where [projName] refers to the project name. 

To edit settings, type - 

`cproj-create config`

Which will open the config_cproj_create.toml file. Please read the 
Instruction to use TOML Files before manually editing yourself!
